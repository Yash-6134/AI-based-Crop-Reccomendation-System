About the Project

KRUSHIMITRA: Smart Crop Recommendation System

This project leverages machine learning to recommend optimal crops based on soil nutrient content and environmental conditions.

Agriculture plays a critical role in India's economic development and is the backbone of our agrarian economy. By using machine learning to recommend crops, we can increase agricultural productivity and sustainability. Our system analyzes soil composition (Nitrogen, Phosphorus, Potassium), weather conditions (temperature, humidity, rainfall), and soil pH to recommend the crop most likely to thrive in those conditions.

How It Works

Our system uses a Random Forest classification algorithm trained on a comprehensive dataset of crop growth conditions. This model identifies complex patterns in the data to provide accurate crop recommendations.

Technical Implementation
  • Machine Learning Model: Random Forest Classifier with hyperparameter tuning
  • Web Application: Flask framework with responsive Bootstrap UI
  • Data Preprocessing: MinMaxScaler and StandardScaler for feature normalization
  • Evaluation Metrics: Classification accuracy, cross-validation, detailed classification reports

Data Sources

The model is trained on the Crop Recommendation Dataset containing 2200 samples with the following attributes:

  • Nitrogen (N) kg/ha
  • Phosphorus (P) kg/ha
  • Potassium (K) kg/ha
  • Temperature °C
  • Humidity %
  • pH pH
  • Rainfall mm
