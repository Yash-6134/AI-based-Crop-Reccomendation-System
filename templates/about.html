{{ _('About the Project') }}

{{ _('KRUSHIMITRA: Smart Crop Recommendation System') }}

{% trans %} This project leverages machine learning to recommend optimal crops based on soil nutrient content and environmental conditions. {% endtrans %}

{% trans %} Agriculture plays a critical role in India's economic development and is the backbone of our agrarian economy. By using machine learning to recommend crops, we can increase agricultural productivity and sustainability. Our system analyzes soil composition (Nitrogen, Phosphorus, Potassium), weather conditions (temperature, humidity, rainfall), and soil pH to recommend the crop most likely to thrive in those conditions. {% endtrans %}

{{ _('How It Works') }}

{% trans %} Our system uses a Random Forest classification algorithm trained on a comprehensive dataset of crop growth conditions. This model identifies complex patterns in the data to provide accurate crop recommendations. {% endtrans %}
